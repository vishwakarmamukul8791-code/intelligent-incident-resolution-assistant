2-POINTER REVISION NOTES (INTERVIEW QUICK SHEET)

1. Two Sum II (Sorted)
   Pattern: Opposite pointers
   Logic:
* sum < target → left++
* sum > target → right--
  Mistakes:
* Using hashmap instead of 2 pointers
* Forgetting array is sorted

---

2. 3Sum
   Pattern: Sort + fix i + 2 pointers
   Logic:

* Fix index i
* Solve 2Sum using left/right
* Skip duplicates at i, left, right
  Mistakes:
* Not skipping duplicates
* Wrong pointer movement after finding answer

---

3. 4Sum
   Pattern: 2 loops + 2 pointers
   Logic:

* Fix i, j → apply 2 pointers
  Mistakes:
* Not handling duplicates at all levels
* No pruning → TLE

---

4. Container With Most Water
   Pattern: Opposite pointers (greedy)
   Logic:

* Move pointer with smaller height
  Mistakes:
* Moving larger height pointer

---

5. Trapping Rain Water
   Pattern: Two pointers + max tracking
   Logic:

* Maintain leftMax and rightMax
  Mistakes:
* Calculating width (not needed)
* Updating max after calculation (wrong order)

---

6. Valid Palindrome
   Pattern: Skip invalid + compare
   Logic:

* Ignore non-alphanumeric
* Compare lowercase characters
  Mistakes:
* Not handling case sensitivity
* Checking same pointer twice

---

7. Valid Palindrome II
   Pattern: At mismatch → skip one char
   Logic:

* Try skipping left OR right
  Mistakes:
* Trying only one possibility

---

8. Is Subsequence
   Pattern: One-directional pointers
   Logic:

* Move pointer in s only when match found
  Mistakes:
* Moving both pointers every time

---

9. Remove Duplicates (Sorted Array)
   Pattern: Slow-Fast pointer
   Logic:

* Slow tracks unique index
  Mistakes:
* Returning wrong length

---

10. Remove Element
    Pattern: Overwrite using slow pointer
    Logic:

* Copy only valid elements
  Mistakes:
* Using extra space unnecessarily

---

11. Move Zeroes
    Pattern: Stable overwrite
    Logic:

* Move non-zero elements forward
* Fill remaining with zero
  Mistakes:
* Swapping unnecessarily

---

12. Sort Colors (Dutch National Flag)
    Pattern: 3 pointers (low, mid, high)
    Logic:

* 0 → swap low, mid++
* 1 → mid++
* 2 → swap high, do NOT mid++
  Mistakes:
* Incrementing mid after swapping with high

---

13. Merge Sorted Array
    Pattern: Fill from back
    Logic:

* Start from last index
  Mistakes:
* Filling from front (overwrites values)

---

14. Squares of Sorted Array
    Pattern: Compare ends
    Logic:

* Fill from back using larger square
  Mistakes:
* Sorting again (O(n log n))

---

15. Backspace String Compare
    Pattern: Traverse from right
    Logic:

* Use counter to skip characters
  Mistakes:
* Using stack (extra space)

---

MASTER TEMPLATES

Opposite Pointer:
while (left < right) {
if (condition) left++;
else right--;
}

Fast-Slow Pointer:
int slow = 0;
for (int fast = 0; fast < n; fast++) {
if (condition) {
nums[slow] = nums[fast];
slow++;
}
}

---

INTERVIEW CHECKLIST

* Sorted array → use 2 pointers
* Pair/Triplet → fix + 2 pointers
* In-place modification → slow-fast
* Ends comparison → opposite pointers

END OF NOTES
